Windows Azure Tools For Microsoft Visual Studio

To enable support for building scalable web applications and services on Windows Azure, install the Windows Azure Tools for Microsoft Visual Studio.

Windows Azure Tools for Microsoft Visual Studio extends Visual Studio to enable the creation, building, debugging, running and packaging of scalable web applications and services on Windows Azure. Installation includes the Windows Azure SDK.

Please see the download page for system requirements, supported operating systems, and installation instructions.

Note: The project that contains this page only provides instructions to install the required toolset.
